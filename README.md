Age Calculator.

Aplicación web que calcula la edad a partir de la fecha de nacimiento ingresada. Proyecto basado en un reto de Frontend Mentor y desarrollado con HTML, CSS y JavaScript.


Prueba en vivo.

https://madisonsosa.github.io/Age-Calculator/


Contenido del repositorio.

index.html
style.css
script.js

Funcionalidades.

Calcula edad en años, meses y días.
Valida fechas incorrectas o vacías.
Muestra mensajes cuando la fecha no es válida.

Funcionalidades extra.

Solo permite el ingreso de números en los campos, ignora letras o símbolos incluso si el usuario intenta escribirlo.
Bloquea automáticamente la entrada cuando ya se han ingresado 2 dígitos en los campos de día y mes, de igual manera en el campo año cuando ya se han ingresado los 4 dígitos.

Interfaz responsiva.

Diseño centrado y limpio.
Compatible con dispositivo móvil y escritorio.

Como usarlo.

Ingresar la fecha de nacimiento usando los campos del formulario.
La aplicación solo aceptara números y detendrá la escritura cuando se complete la cantidad correcta de dígitos.
Presionar el botón con la flecha hacia abajo (↓) para realizar el cálculo.
Ver la edad calculada en pantalla.
